## 本地開發

1. Clone 專案
```bash
git clone https://github.com/henryli-ai/chatgpt-enterprise.git
cd chatgpt-enterprise
```

2. 使用任何 HTTP 伺服器啟動
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx http-server

# 或直接在瀏覽器開啟 index.html
```

3. 在瀏覽器中訪問 `http://localhost:8000`

## 部署
